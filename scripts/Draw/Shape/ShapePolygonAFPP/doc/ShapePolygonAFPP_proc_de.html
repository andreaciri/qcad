1. Geben Sie den Grad des Polygons (Anzahl Ecken / Kanten) in der Optionenwerkzeugleiste ein. Gültige Zahlen sind 3 bis 99.
  2. Wählen Sie die Option Ecke zu Ecke um das Polygon über zwei gegenüberliegende Ecken zu definieren.
  3. Bestimmen Sie die erste Ecke oder Mitte einer Seite des Polygons mit der Maus oder geben Sie eine Koordinate in die Kommandozeile ein.
  4. Bestimmen Sie die zweite, gegenüberliegende Ecke oder Mitte einer Seite des Polygons.
