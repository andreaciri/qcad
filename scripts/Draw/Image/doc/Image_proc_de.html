1. Wählen Sie die Bilddatei, die Sie einfügen wollen.
  2. Geben Sie einen Skalierungsfaktor in die Optionenwerkzeugleiste ein. Ein Faktor von 1.0 heisst, dass ein Pixel des Bildes einer Einheit in der Zeichnung entspricht.
  3. Geben Sie den Rotationswinkel in die Optionenwerkzeugleiste ein.
  4. Bestimmen Sie die Position des Bildes. Das ist die linke untere Ecke des Bildes.
