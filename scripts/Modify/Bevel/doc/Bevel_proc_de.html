1. Geben Sie die Geometrie der Abschrägung in der Optionenwerkzeugleiste ein. 'Länge 1' ist der Abstand der Abschrägung vom Schnittpunkt der beiden Objekte auf dem ersten Objekt. 'Länge 2' ist der gleiche Abstand für das zweite Objekt.
  2. Aktivieren Sie 'Trimmen' wenn Sie die beiden Objekte der Ecke automatisch bis zur Abschrägung trimmen wollen. Wenn diese Option nicht angewählt wird, bleiben die ursprünglichen Objekte unverändert.
  3. Wählen Sie das erste Objekt (eine Linie oder ein Kreisbogen).
  4. Wählen Sie das zweite Objekt.
