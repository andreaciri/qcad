1. Wählen Sie das limitierende Objekt zu dem ein oder mehrere Objekte getrimmt werden sollen.
  2. Wählen Sie die Objekte, die Sie zum limitierenden Objekt trimmen wollen. Oft gibt es zwei Möglichkeiten wie ein Objekt getrimmt werden kann. In der Abbildung wird der obere Teil der Linie getrimmt. Soll stattdessen der untere Teil der Linie getrimmt werden, muss die Linie am oberen Teil angeklickt werden. Immer derjenige Teil eines Objektes, der angeklickt wird bleibt erhalten.
  3. Klicken Sie die rechte Maustaste oder drücken Sie zweimal Escape um das Werkzeug zu beenden.
