Toggles the linetype mode of the current drawing.
If the linetype mode is activated, linetypes are optimized for a computer screen. In this mode, all lines are displayed with a screen based line width (lines don't get wider when zooming in) and all patterns are shown as fixed, pixel based patterns on screen (dashes don't get longer when zooming in).
If the linetype mode is off, line widths and linetypes are shown in drawing units (default).
