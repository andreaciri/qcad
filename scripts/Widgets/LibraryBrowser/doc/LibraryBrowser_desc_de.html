Die Symbolbibliothek von QCAD ist eine Sammlung von Symbolen und anderen Zeichnungen die in Ihren Zeichnungen wiederverwendet werden können. Um eine Zeichnung aus der Symbolbibliothek einzufügen, müssen Sie zuerst den Library Browser anzeigen.
Die Symbole können per Drag und Drop in die Zeichnung eingefügt werden. Sobald der Mauszeiger innerhalb der Zeichenfläche ist, können Sie die regulären Fangwerkzeuge zum Positionieren verwenden. Mit Hilfe der Optionenwerkzeugleiste können Sie das Symbol ausserdem skalieren, drehen oder rotieren.
