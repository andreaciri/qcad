Dieses Werkzeug exportiert die aktuelle Zeichnung in eine PDF-Datei. Der PDF Export einer Zeichnung ist sehr ähnlich wie das Drucken einer Zeichnung. Sie werden höchstwahrscheinlich die Skalierung und Position Ihrer Zeichnung anpassen müssen vor dem PDF Export.
Aus diesem Grund schaltet QCAD automatisch die Druckvorschau ein wenn dieses Werkzeug gestartet wird. Wenn die Vorschau nicht Ihrem Wunsch entspricht, brechen Sie den Export ab und richten Sie erst die Seite ein.
