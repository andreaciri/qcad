1. Starten Sie dieses Werkzeug wenn Sie einen Punkt definieren wollen indem Sie dessen Koordinate eingeben.
  2. Geben Sie die gewünschte Koordinate in der Optionenwerkzeugleiste ein und wählen Sie ob es ein relative oder absolute Koordinate ist. Relative Koordinate beziehen sich auf die Position des relativen Nullpunktes.
