1. Verwenden Sie die Auswahlwerkzeuge, um die Objekte, die Sie kopieren möchten auszuwählen.
  2. Starten Sie das Kopierwerkzeug.

Die Kopierfunktion von QCAD funktioniert wie die Kopierfunktion von anderen Anwendungen. Als Bezugspunkt wird der automatisch berechnete Mittelpunkt der Auswahl verwendet. Dies ist nicht immer wünschenswert, da Sie meistens die eingefügten Objekte an einer genau definierten Position platzieren wollen. Hierfür bietet QCAD eine alternative, CAD-spezifische Funktion an: Kopieren mit Referenz.

Die Auswahl ist nun auf der QCAD-internen Zwischenablage und kann in die gleiche oder eine andere Zeichnung eingefügt werden. Die Zeichnung in die eingefügt wird muss mit der gleichen Instanz von QCAD geöffnet werden in der auch kopiert wurde. Wenn QCAD beendet wird, geht der Inhalt der Zwischenablage verloren. Auf die QCAD Zwischenablage kann nur von QCAD selbst zugegriffen werden. Sie können mit Kopieren und Einfügen keine Inhalte mit anderen Applikationen austauschen. Benutzen Sie dafür stattdessen die Import- und Exportwerkzeuge von QCAD.
