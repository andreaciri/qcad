Cross Selection

Some selection tools can be used in cross selection mode. In this mode, not only entities that are completely inside a given area are selected but also entities that are only partly inside the area. This selection is also known as 'cross selection'.

Selection Mode

Some selection tools allow you to choose a selection mode in the options tool bar. The selection modes that are available are:

  • Replace selection:

    Replaces the current selection with the new selection made by the active selection tool (default).
  • Add to selection:

    Adds the selection to the current selection.
  • Remove from selection:

    Removes (subtracts) the selection from the current selection.
  • Intersect:

    Only entities that were already selected and that match the criteria of the selection tool are selected.
